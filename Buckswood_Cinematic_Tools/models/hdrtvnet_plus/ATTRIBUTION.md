# HDRTVNet++ Attribution

`AGCM.pth` is the pretrained Adaptive Global Color Mapping model from:

- Xiangyu Chen et al.
- "Towards Efficient SDRTV-to-HDRTV by Learning from Image Formation"
- https://github.com/xiaom233/HDRTVNet-plus

The upstream code and model repository are distributed under the MIT License.
The original license is included as `LICENSE_HDRTVNet-plus`.

The Buckswood adapter reimplements only the inference architecture required to
load the upstream AGCM state dictionary. It does not include the upstream
training datasets or copyrighted source videos.
