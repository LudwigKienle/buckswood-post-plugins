Buckswood Cinematic Tools v2.1
================================

Includes:

- Buckswood Frame Director v2.1
- Buckswood Radiance Recover v2.1
- Buckswood Temporal Integrity v2.1

Installation:

1. Quit DaVinci Resolve.
2. Open Buckswood_Cinematic_Tools_Installer.pkg.
3. Complete the installer.
4. Restart Resolve.
5. Open Color > Effects > Buckswood.

Radiance Recover v2 can use a local HDRTVNet++ ML cache. The ML runtime remains
outside Resolve so a missing model or cache cannot destabilize the host.
