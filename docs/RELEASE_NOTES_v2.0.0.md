# Buckswood Post Plugins v2.0.0

First public multi-host release candidate.

## DaVinci Resolve

Stable v2 package for macOS and Windows.

Included:

- Buckswood Fake Diagnostic v2.1
- Buckswood Lens Physics v0.3 Anti-Ghosting
- Buckswood AI Photorealizer v0.2

Assets:

- `Buckswood_Resolve_Plugins_v2_macOS.dmg`
- `Buckswood_Resolve_Plugins_v2_macOS.zip`
- `Buckswood_Resolve_Plugins_v2_Windows_Setup.exe`
- `Buckswood_Resolve_Plugins_v2_Windows.zip`

## Nuke

Experimental OFX host package.

Included:

- Buckswood Fake Diagnostic v2.1
- Buckswood Lens Physics v0.3 Anti-Ghosting
- Buckswood AI Photorealizer v0.2

Asset:

- `Buckswood_Nuke_OFX_v2.zip`

## Premiere Pro

Native beta port.

Included:

- Buckswood Lens Physics
- Buckswood AI Photorealizer

Assets:

- `Buckswood_Premiere_Native_macOS.zip`
- `Buckswood_Premiere_Native_Windows.zip`
- `Buckswood_Premiere_Plugins_Setup.exe`

Fake Diagnostic is not included in the Premiere package yet because it needs a dedicated native temporal/diagnostic UI pass.

## Signing Notes

- Resolve macOS release is signed and notarized.
- Nuke macOS OFX bundles are Developer-ID-signed.
- Premiere macOS bundles are Developer-ID-signed.
- Windows setup apps are not Authenticode-signed yet, so Windows SmartScreen may warn.
