Buckswood Lens Physics for DaVinci Resolve
==========================================

Installation:

1. Close DaVinci Resolve.
2. Double-click Buckswood_Lens_Physics_Installer.pkg.
3. Finish the macOS Installer steps.
4. Restart DaVinci Resolve.

Resolve path:

Color Page > OpenFX > Buckswood > Buckswood Lens Physics

DCTL fallback:

Color Page > DCTL > Buckswood_Lens_Physics_v01

Installed files:

/Library/OFX/Plugins/BuckswoodLensPhysics.ofx.bundle
/Library/Application Support/Blackmagic Design/DaVinci Resolve/LUT/Buckswood_Lens_Physics_v01.dctl

If macOS says the package is from an unidentified developer, right-click the
package and choose Open. This package is unsigned unless you sign it with your
Apple Developer ID.
