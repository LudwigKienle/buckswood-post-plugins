# Lensfun Data Attribution

This project can use lens calibration data from the Lensfun project.

Source:
https://github.com/lensfun/lensfun

Local source checkout:
`assets/lensfun_repo/data`

License for the Lensfun lens database:
Creative Commons Attribution-ShareAlike 3.0 Unported, CC BY-SA 3.0.

Local license copy:
`assets/lensfun_repo/data/COPYING.CC_BY-SA_3.0`

Important redistribution note:
If Buckswood Lens Physics ships with Lensfun-derived lens profile data, the distributed lens profile data and adaptations of it must preserve attribution and follow the CC BY-SA 3.0 terms. Do not mix in commercial or proprietary lens-profile data unless the license explicitly allows redistribution.
