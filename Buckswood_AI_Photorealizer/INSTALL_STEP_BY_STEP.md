# Buckswood AI Photorealizer Install Step by Step

## Normale Plugin-Installation

Nimm fuer normale Nutzer die DMG:

```text
/Users/ludwig.kienle/Downloads/Davinci_plug_in/Buckswood_AI_Photorealizer/release/Buckswood_AI_Photorealizer_Installer.dmg
```

1. DaVinci Resolve komplett schliessen.
2. DMG doppelklicken.
3. `Buckswood_AI_Photorealizer_Installer.pkg` doppelklicken.
4. macOS Installer durchklicken.
5. Admin-Passwort eingeben, wenn macOS fragt.
6. Resolve neu starten.

Falls macOS meckert: Rechtsklick auf das `.pkg`, dann `Oeffnen`.

## In DaVinci Resolve finden

OFX-Version:

```text
Color Page > OpenFX > Buckswood > Buckswood AI Photorealizer
```

DCTL-Fallback:

```text
Color Page > DCTL > Buckswood_AI_Photorealizer_v01
```
